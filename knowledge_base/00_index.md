# Индекс документов базы знаний

> Начальный документ для RAG-системы проекта Alsafi.

## Как добавлять документы

1. Создайте новый `.md` файл в этой папке
2. Используйте чёткие заголовки (`##`, `###`)
3. Пишите структурированно: разделы, списки, код — это упрощает чанкинг

## Пример структуры документа

```markdown
# Название темы

## Раздел 1
Содержание...

## Раздел 2
Содержание...
```
