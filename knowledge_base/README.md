# База знаний для RAG

Эта папка содержит документы для системы Retrieval-Augmented Generation (RAG).

## Структура

- Документы хранятся в формате **Markdown** (`.md`) — их удобно разбивать на чанки для эмбеддингов
- Имена файлов должны быть осмысленными (например, `topic_name.md`)
- Для лучшего поиска добавляйте заголовки и ключевые слова в начале документа

## Использование с RAG

Типичный пайплайн:

1. **Загрузка** — чтение `.md` файлов из `knowledge_base/`
2. **Чанкинг** — разбиение на фрагменты (например, по 500–1000 символов с overlap)
3. **Эмбеддинги** — векторизация через OpenAI, HuggingFace и т.п.
4. **Индексация** — сохранение в векторной БД (ChromaDB, FAISS, Pinecone)
5. **Поиск** — семантический поиск по запросу пользователя и подстановка релевантных фрагментов в промпт LLM

## Рекомендуемые зависимости

```
langchain
chromadb
sentence-transformers  # или openai для эмбеддингов
```
