# DRM (Django)

## Запуск

Из каталога `drm/`:

```bash
# Только с этой машины (по умолчанию)
python manage.py runserver

# Доступ с других машин по сети (например http://172.25.95.61:8000/)
python manage.py runserver 0.0.0.0:8000
```

Если с другой машины не открывается даже `http://IP:8000/health/` — убедитесь, что запущен `runserver 0.0.0.0:8000` и порт 8000 не блокируется файрволом.

## SSO (Windows Auth)

Вход только по заголовку пользователя от IIS/прокси. Для локальной разработки без IIS задайте в `.env` в корне проекта:

```
SSO_DEV_USER=ваш_логин
```
